## Features

- Modern responsive UI  
- Trending & top-rated movies  
- Search functionality  
- Watchlist support  
- WebView video player  
- Light / dark themes  
- Ads integration ready  

---

## Getting Started

### Requirements
- Flutter SDK (stable)
- Android Studio / VS Code

### Setup

```bash
git clone https://github.com/suadatbiniqbal/harmbermovies.git
cd harmbermovies
flutter pub get
flutter run

'''
